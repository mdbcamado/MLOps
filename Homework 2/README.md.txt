# Fruit Classifier Project

This project implements a machine learning model to classify fruits based on their physical characteristics.

## Dataset
The dataset contains measurements of fruits including:
- Mass
- Width
- Height
- Color score

Fruits classified:
- Apple
- Mandarin
- Orange
- Lemon

## Fruit Classifier in streamlit can be accessed through accessing a terminal.

1)Navigate through the working directory

2) Type python -m venv venv
3) venv\Scripts\activate
4) python main.py
